CSS Colors and its Various Representations

Named Colors

CSS includes 147 named colors that allow developers to add color to their websites.

Here are some example named colors:

Color Name Color Represented
Aquamarine
DarkGoldenRod
Lavender
MediumOrchid
Sienna

Hex Value Representations

Colors can be represented using a # symbol followed by six hexadecimal digits. Duplicate digits can be simplified.

Hexadecimal Value Color Represented
#DEBF5B
#A53CD8
#6DF623
#A0BFC8
#77AA00 or #7A0

rgb or rgba Representations

Colors can also be represented by using the rgb attribute. The rgb takes 3 values between 0 and 255 signifying red, green, and blue.

There is also the rgba attribute with the fourth value representing the opacity taking a value between 0 and 1.

rgb or rgba Color Represented
rgb(100, 230, 74)
rgb(179, 29, 123)
rgb(99, 38, 222)
rgb(255, 139, 69)
rgba(188, 86, 205, 0.7)

hsl or hsla Representations

CSS colors can also be represented by using the hsl attribute. The hsl takes three inputs, the first is hue: 0-360, the second is saturation: percentage, and the third is lightness: percentage.

There is also the hsla attribute with the fourth value representing the opacity taking a value between 0 and 1.

hsl or hsla Color Represented
hsl(275, 87%, 60%)
hsl(167, 46%, 80%)
hsl(340, 28%, 95%)
hsl(132, 100%, 22%)
hsla(18, 18%, 18%, 0.5)
